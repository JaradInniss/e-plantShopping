# e-PlantShopping Repository
This repository holds the content for E-Plant Shopping, an online shop for Paradise Nursery plant nursery.
Paradise Nursery is an organization passionate about bringing nature closer to its customers. 
Their mission is to provide a wide range of high-quality plants that beautify customers' surroundings and contribute to healthier and more sustainable lifestyles.

### Website Layout
- Landing page: The Website features a Landing Page that informs users about Paradise Nursery and directs them to the main content of the website.
- Product Listing page: The main content of the website. Features a navigation bar that allows users to navigate to the Landing Page, Plant Listing Page, and Cart        Page. Also features the listing of the plants offered by Paradise Nursery. Plant information is listed (name, price, etc.). Allows users to add plants to a cart.
- Cart Page: Displays the products users added to the cart. Allows users to increase or decrease the number of an item in the cart, or remove it completely. Displays     the total for each item and the overall total.
